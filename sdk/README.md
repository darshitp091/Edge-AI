# EdgeAI Neural SDK

The official Python library for sharding and optimizing machine learning models for edge hardware.

## Installation

```bash
pip install edge-ai-sdk
```

## Quick Start

1. **Authenticate**
   ```bash
   edge-ai auth --login
   ```

2. **Compress a Model**
   ```bash
   edge-ai compress ./model.onnx --target jetson
   ```

## Development

To install in editable mode for local testing:
```bash
pip install -e .
```
